# JRename

JavaFX 11

The purpose of this application is to make it easier to rename episodes of a series, anime or anything like that. 

Instead of renaming each episode manually, you can provide a default file name for all file names as well starting file number. The app will then rename each file accordingly. 

You can also get the correct file names from a source and paste them into the app. The application will then rename each file. However, you must paste per line.
